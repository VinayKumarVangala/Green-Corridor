# Ambulance Driver Guide

## 1. Login
- Use assigned credentials from dispatch control.

## 2. Status Workflow
- `Available` ??? `En route` ??? `Arrived` ??? `On board` ??? `Delivered`.
- Update status via the ambulance dashboard.

## 3. Location Updates
- Keep GPS sharing on.
- Frequent updates allow AI rerouting and traffic clearance.

## 4. Incident Handling
- Confirm pickup and arrival via app actions.
- Notify hospital once patient is on board.
