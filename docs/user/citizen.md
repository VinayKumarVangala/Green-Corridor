# Citizen Guide

## 1. Overview
As a citizen, use the mobile/web interface to request emergency support and track ambulance arrival.

## 2. Requesting Help
1. Open the app and login/continue as guest.
2. Fill emergency details and allow location access.
3. Submit and note the request ID.

## 3. Tracking Ambulance
- Live map tracking.
- ETA updates from the system.
- In-app chat/call to confirm.

## 4. After Care
- Confirm patient boarding.
- Provide feedback.
